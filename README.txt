== Description ==

**MWB MULTI-CURRENCY SWITCHER FOR WOOCOMMERCE ALLOWS SEAMLESS CURRENCY CONVERSION, AND LET'S YOU TO CALCULATE AND CONVERT CURRENCY IN REAL-TIME FOR INTERNATIONAL CUSTOMERS.**

Multi-currency switcher for WooCommerce allows your customers to view the price of the products in their local currency. Hence, expanding your business across boundaries without currency being a barrier becomes easy with the currency switcher plugin. This functionality adds utility to your store to make it more user-friendly in the sense that your visitor does not need to worry about unnecessary calculations whilst shopping. To be able to view the products and services in their currency is a feature that enhances the user experience. This multi-currency switcher for WooCommerce works efficiently because of the 5 added efficient currency aggregators. The additional perk you get here as a merchant is that no API key is needed to use these currency aggregators.

== WHAT’S INCLUDED IN OUR MWB MULTI-CURRENCY SWITCHER PLUGIN: ==

**1.** **EFFICIENT AGGREGATORS:**
Merchants can select out of 5 currency aggregators for effective conversion and real-time exchange rate updation. You need no API key for the same. The plugin also allows you to select the frequency of updation of the exchange rates. 

**2.** **CURRENCY DISPLAY:**
Using the radio button, you get to choose the currency you want to display. You also get the option to decide the currency symbol and its placement for your currency. In the currency switcher plugin, you get the option to set the interest and rates manually or automatically, in separate text boxes. 

**3.** **CUSTOMIZE CURRENCY OPTIONS:**
You can change the currency switcher options and choose to hide the currency switcher option during checkout. Also, the merchant gets to allow the user to pay in the native currency or the default currency.

**4.** **WELCOME CURRENCY:**
You get the option to choose the welcome currency for your visitors. A welcome currency simply means a preset default currency. This default currency makes it convenient for the user to begin shopping from your store right away.

**5.** **CUSTOM PRICE FORMAT:**
You can set the price format according to your choice by placing the price and currency symbol on either of the sides. It is not just an aesthetic component but helps you in taking care of what price can be easily perceived by the user. 

**6.** **CONSISTENT PRICING DISPLAY:**
The plugin will ensure a smooth and consistent display of your pricing choice across the complete WooCommerce store. 
 
**7.** **REPRESENTATION:**
The currency switcher plugin allows you to style the switcher in a dropdown menu in three formats- classic block, roll block, and round select. You can select an option to enhance the aesthetic utility of the currency switcher plugin on your store. With the latest version, the selected currency will be highlighted in the side switcher menu to switch currency.  


**8.** **DETECTION WITH LOCATION:**
The merchant can set different GeoIP rules for the detection of the visitors’ currency according to their location. When a visitor visits your online store for the first time, their IP address is tracked. With the help of the defined GeoIP rules, the prices are automatically displayed in the native currency. 

**9.** **CUSTOMIZE PUBLIC PANEL:**
You can define the front-end features you want to provide individually for the product page module, cart page module, checkout page module, and mini-cart module. Thus, the currency switcher plugin also empowers you to display the native currency of the shipping amount of your users’ orders.

== Frequently Asked Questions ==

= How much currency does an admin update in the plugin? =

Admin can update and use three international currencies for now with our plugin “MWB Multi-Currency Switcher for WooCommerce” , if you want to use more than three currencies then please visit our pro version.

= Is the admin able to set one currency as default for other currencies? =

Yes, the admin can set anyone currency as default from the currency setting in the plugin backend and this will also provide the values to other currencies too.

     
= Is the admin able to put a rate of interest for interchanging the currencies? =

Yes, the admin can set the interest for interchanging the multi-currencies for an order from the rate & interest tab in the currency setting of the plugin backend.


= Is there a way to set any welcome currency for the clients so that only the selected currency can reflect the customer? =

Yes, the admin can set the welcome currency for the clients from the advanced setting in the plugin backend, after which the client can also convert the currency to its choice from the available currency switcher widget.


= Is there any setting through which the admin can set the currencies for a set of countries so that the client can automatically get the desired currency on the site? =

Yes, the admin can set the currency for selected nations from the Geolocations Rules tab in the plugin backend and enable this setting from the advanced setting provided in the plugin.


= How would the admin be able to put the currency switcher on the frontend or on the shop page? =
 
Admin can use the currency switcher widget in the sidebar in the widget setting >>appearances.


= Is there any shortcode for currency converters? =

Yes, the admin can use the [currency_converter] shortcode which will allow you to convert currency and see the value of the currency in real-time.


= Is there any shortcode for the currency switcher dropdown to put specifically in any place? =

Yes, the admin can use the [currency_switcher_dropdown] shortcode which will add currency dropdown and selected currency flag and will also change the value of the page where it is applied.
